# Marvin Go

A lightweight, voice-first mobile client for [Amazing Marvin](https://amazingmarvin.com).

## Features

- **Voice Interaction:** Add tasks, manage projects, and ask about your schedule using Gemini-powered voice commands.
- **Mobile Optimized:** Designed as a Progressive Web App (PWA) for a native-like experience on your phone.
- **Offline Capable:** Works offline and syncs when back online (PWA capabilities).
- **Focus Mode:** Simple interface to keep you focused on the current task.

## Installation

1. Open the app in your browser.
2. Tap "Share" (iOS) or the menu button (Android).
3. Select "Add to Home Screen".

## Development

```bash
npm install
npm run dev
```
